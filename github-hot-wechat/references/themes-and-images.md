# 联动主题与图片规则

| 主题 | 视觉 | 用途 |
|---|---|---|
| `open-coordinates` | 暖白、薄荷、深青、暖橙；开放坐标与路线 | 默认探索感 |
| `code-archive` | 纸张、索引标签、蓝灰、琥珀 | 技术与资料感 |
| `field-notes` | 温暖纸张、印刷线条、手账结构 | 观察笔记感 |
| `clean-grid` | 大留白、细网格、蓝绿、高亮色 | 清爽工具感 |

`--theme auto` 按运行周稳定轮换；同一运行日期重复执行必须得到同一主题。

图片交付：1283×383 合并封面、900×383 横版封面、383×383 方形封面、每个项目一张 1200×675 横图及一张 1200×675 结尾图。单张小于 2 MB。

Image 2 提示词必须要求无文字、无日期、无 Logo、无水印、无真实产品界面，并为标题与裁切预留安全区。准确中文由本地排版完成。没有 Image 2 时使用对应主题的模板图继续生成，在运行报告标记 `template_fallback`，不因此降低内容审核状态。
